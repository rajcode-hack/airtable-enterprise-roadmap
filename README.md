
# Enterprise Airtable System

An enterprise-grade Airtable-inspired roadmap and visualization platform built using React and Node.js.

This project demonstrates real-world experience in:
- Airtable visualization
- Wireframing-driven development
- Product roadmap communication
- Design-to-engineering handoff

---

## Features

- Enterprise dashboard
- Workspaces, Programs, Members modules
- Airtable-style relational data
- 100+ seeded records
- React frontend with routing
- Node.js backend APIs

---

## Tech Stack

Frontend:
- React
- React Router
- Enterprise UI styling

Backend:
- Node.js
- Express
- REST APIs
- Seeded enterprise data
